# Lyrics Data

## Requirements

### Libraries

- Python >=3.14
- uv (see [Install uv](https://docs.astral.sh/uv/getting-started/installation/))

### Install libraries

```bash
uv sync
```

## Count occurrences

```bash
uv run main.py
```

## Download lyrics

```bash
uv run main.py -d
```

## Clear results

```bash
uv run main.py -c
```
